# 通知表所見自動生成ツール

小学校の通知表所見を自動生成するWebアプリケーションです。

## 機能

- キーワード入力から自然な日本語の所見文を自動生成
- 学習面・生活面合わせて指定文字数（デフォルト200文字）で生成
- 作成した所見の一覧管理
- QRコードで簡単に共有

## セットアップ

### 1. 依存関係のインストール

```bash
pip install -r requirements.txt
```

### 2. 環境変数の設定

`.streamlit/secrets.toml` ファイルを作成し、以下を設定：

```toml
OPENAI_API_KEY = "your-api-key-here"
```

### 3. アプリの実行

```bash
streamlit run app.py
```

## 使い方

1. キーワードを入力（または選択）
2. 文字数を指定（デフォルト200文字）
3. 「所見を生成」ボタンをクリック
4. 生成された所見を確認・保存

## デプロイ（重要）

**他の先生にも使ってもらうには、Streamlit Cloudにデプロイする必要があります。**

ローカル環境では、同じWi-Fiネットワーク内のデバイスからのみアクセス可能です。
Streamlit Cloudにデプロイすることで、どこからでもアクセスできるようになります。

### デプロイ手順

詳しい手順は **`DEPLOY_GUIDE.md`** を参照してください。

**簡単な手順:**
1. GitHubにリポジトリをプッシュ
2. Streamlit Cloudでアプリをデプロイ（無料）
3. Streamlit Cloudのダッシュボードで `OPENAI_API_KEY` を設定
4. QRコードを配布して、どこからでもアクセス可能に
